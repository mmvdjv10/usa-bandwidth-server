## 二、先搞清楚：什么是"大带宽"，多大才算大

"大带宽"这个说法其实没有官方标准。但从业内通用的认知来看，大致可以这样划分：

- **300Mbps**：入门级独享带宽，适合中小流量站点、企业官网、轻量级应用
- **1Gbps**：主流大带宽，能扛住大部分电商、内容站、中型社区
- **2Gbps**：中高带宽，适合视频站、下载站、CDN节点
- **10Gbps**：超大带宽，面向高流量业务、大流量分发、重度应用

这里有个关键概念必须分清楚——**计量带宽**和**不限流量带宽**。

计量带宽就是给你一个流量额度，比如每月10T，超了要么限速要么加钱。不限流量带宽则是只要不超过端口速率上限，跑多少都行，不会因为流量超额产生额外费用。

对于流量波动大的业务，不限流量带宽的价值就体现出来了。你不用担心某天突然来一波流量把账单炸掉。

GTHost 全线产品都是不限流量带宽，这一点在选购时要重点留意——同样是"1Gbps"，一家计量一家不限量，实际成本差别可能很大。
